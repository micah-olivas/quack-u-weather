# quack-u-weather

I built the Quack-u-weather web app to serve as a front end for a predictive regression model. A gradient-boosted machine algorithm (GBM via Caret package in R) was trained on 12 meteorological predictors from more than 390 observations of daily hunting outcomes at a wetlands preserve in Wasco, CA.
